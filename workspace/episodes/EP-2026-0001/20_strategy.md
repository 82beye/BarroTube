## 4. 톤앤매너

### 말투
- **친근한 해설체**: "~입니다", "~인데요", "~거든요" 혼용
- **대화형 질문 삽입**: "여러분, 이거 아셨나요?", "혹시 궁금하지 않으세요?"
- **숫자의 일상 치환**: "GDP 1.7% 증가는 여러분 월급에서 약 X만 원 차이입니다"
- **반전 장치**: "근데 여기서 중요한 게 있습니다", "그런데 반대로 생각해보면"

### 화자 포지셔닝
- "어려운 경제를 쉽게 풀어주는 친구" (brand.md 준수)
- 전문가 행세 X → 공인 기관 데이터를 대신 정리해주는 역할
- "저도 여러분처럼 궁금해서 찾아봤는데요" 식의 동료 시점

### 금기어 (brand.md 기준)
| 금기어 | 대체 표현 |
|--------|----------|
| "무조건 오른다/떨어진다" | "상승/하락 가능성이 있습니다" |
| "지금 당장 사세요/파세요" | 사용 금지 (투자 권유 불가) |
| "100% 확실" | "데이터에 따르면", "전문가들은 ~로 전망합니다" |
| "이것만 하면 부자" | 사용 금지 |
| 특정 종목 매수/매도 추천 | "AI 수혜 산업군" 수준에서 언급, 종목명 불가 |
| 정치적 편향 발언 | 정책은 객관적 수치 소개에 한정 |
| "사라지는 직업" | "변화하는 직업 지형", "역할이 재정의되는 분야" |

### 선호 표현
- "데이터로 보면", "숫자가 말해줍니다"
- "변화의 방향", "기회의 영역"
- "IMF/맥킨지/OECD에 따르면"
- "한국으로 좁혀보면", "우리 경제에 대입하면"
- "정리하면 이렇습니다", "핵심만 짚어보겠습니다"

## 6. CTA (Call to Action)

### 영상 중간 CTA (본론2 종료 직전, ~5:30 지점)
> **나레이션**: "이런 변화의 흐름이 궁금하시다면, 구독과 알림 설정 해두시면 매일 핵심만 정리해서 전해드리겠습니다."
>
> **화면**: 우측 하단에 구독 버튼 애니메이션 (3초간 표시 후 페이드아웃)
>
> **목적**: 시청 지속 중 자연스러운 구독 유도. 강압적이지 않은 "정보 제공자"로서의 포지셔닝 유지.

### 영상 끝 CTA (아웃트로, ~7:40 지점)
> **나레이션**: "오늘 영상이 도움이 되셨다면 좋아요 한 번 부탁드리고요. 댓글로 여러분이 가장 궁금한 경제 주제를 알려주세요. 다음 영상에서 다뤄보겠습니다. 내일도 핵심만 짧게, 오늘의 경제 브리핑에서 만나요."
>
> **화면**: 좌측에 좋아요/구독 버튼, 우측에 다음 영상 프리뷰 카드(엔드 스크린), 하단에 면책 조항 자막
>
> **목적**: 좋아요(알고리즘 부스트) + 댓글(참여율 향상) + 시리즈 유도(채널 체류 시간 증가)
